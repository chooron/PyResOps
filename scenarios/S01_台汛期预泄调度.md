# S01 台汛期预泄调度场景

## 场景概述

**场景名称：** 台汛期台风来临前预泄腾库  
**季节背景：** 台汛期（7月1日～10月15日）  
**核心目标：** 预报台风影响前，将库水位从 157.5m 降至汛限水位 156.5m，腾出防洪库容  
**参考文件依据：** 2.5节防洪调度原则 第①②条、4.2.2节泄洪放水预警原则

## 验证要点

1. **水量平衡验证：** 48小时末水位应接近 156.5m
2. **约束校核验证：** 全程出库流量不超过下游安全泄量
3. **爬坡约束验证：** 流量变化不超过 500 m³/s/步
4. **生态约束验证：** 任何时刻出库不低于 50 m³/s
5. **决策追踪验证：** trace 中应记录规则命中和预泄原因

## MCP 调用链

```
1. get_reservoir_snapshot()           → 获取当前水位快照
2. generate_dispatch_program()        → 生成预泄调度方案
3. simulate_program(with constraints) → 仿真执行并校核约束
4. evaluate_program()                 → 评估防洪得分和发电效益
```
