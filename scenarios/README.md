# 滩坑水电站调度场景库

本目录基于《浙江省瓯江滩坑水电站2025年度水库控制运用计划》设计，针对 PyResOps 库的核心能力进行验证，同时为大模型调用提供典型场景参考。

## 能力覆盖评估

PyResOps 当前能力与运控计划需求的对应关系：

| 运控计划需求 | PyResOps 覆盖能力 | 覆盖状态 |
|-------------|-----------------|---------|
| 水量平衡仿真 | `SimulationEngine` + 水量平衡推进 | ✅ 完整覆盖 |
| 防洪约束校核 | `level_max/min/range`, `flow_max` | ✅ 完整覆盖 |
| 下游错峰调度 | `downstream_flow_limit` 约束 | ✅ 完整覆盖 |
| 生态流量控制 | `ecological_min_flow` 约束 | ✅ 完整覆盖 |
| 预泄调度规则 | `ExpressionRuleEvaluator` 表达式规则 | ✅ 完整覆盖 |
| 多目标评估 | `flood/supply/power/ecology/compliance` 指标 | ✅ 完整覆盖 |
| 滚动调度优化 | `RollingOpsService` | ✅ 完整覆盖 |
| 决策追踪回放 | `decision_traces` 持久化 | ✅ 完整覆盖 |
| 分期汛限水位 | `PolicyBundle` 规则动态切换 | ✅ 可通过规则实现 |
| 区间洪水预报 | 需外部输入 `forecast` 数据 | ⚠️ 需外部数据支持 |
| 闸门开度精细控制 | 当前以流量为控制量 | ⚠️ 间接支持 |
