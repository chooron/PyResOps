# 1 基本情况

# 1.1 水库概况

滩坑水电站位于青田县境内小溪干流下游，坝址以上流 域面积 $3 3 3 0 \mathrm { k m } ^ { 2 }$ ，占小溪流域总面积的 $9 3 \%$ 。滩坑水电站总装 机容量 $6 0 4 \mathrm { M W } \mathrm { ~ ( ~ 3 ~ \times ~ 2 ~ 0 ~ 0 { M W } + 4 { M W } ~ ) ~ }$ ，水库具有多年调节性能。滩 坑水电站担负浙江省电力系统调峰、调频、调相及事故备用 任务，同时兼顾防洪，并具有其他综合利用效益。滩坑水电 站水库校核洪水位169.15m，设计洪水位165.87m，总库容41.9 亿 ${ \mathfrak { m } } ^ { 3 }$ ；正常蓄水位 160.00m，相应库容 35.2 亿 m3 ；防洪高水 位 161.50m，梅汛期限制水位 160.00m，台汛期限制水位 156.50m，防洪库容 3.5 亿 m3 ；死水位 120.00m，调节库容 21.26 亿 ${ \mathfrak { m } } ^ { 3 }$ 。电站设计保证出力 87.75MW，多年平均年发电量 10.23 亿 kWh，装机年利用小时数1694h。 

滩坑流域内略大的支流有左岸的英川溪、右岸的标溪及 炉西坑等。暴雨的成因是春末夏初（5～6 月）暖湿太平洋高 压气团渐向大陆推进，极峰常在此停滞或摆动，造成连续降 水，俗称梅雨季节；盛夏季节（7～9 月）在副热带高压脊控 制下，天气炎热，盛行偏南风，且台风频繁，多降热雷雨和 台风雨，台风对本流域影响较大，有时一次台风雨甚至超过 常年的月降水量。发生在本流域的台风雨，其雨量分布是上 游小，下游大，暴雨中心常出现在流域下游左侧的小佐附近， 台风雨日降水量最大可达 300mm 以上，历时较短，一般1～2 天。峰面雨雨量分布则相反，多是上游大，下游小，暴雨中 心多在流域上游湖坪头、贤良一带，峰面雨日降水量不大， 均在 150mm 以下，但历时较长。本流域洪水由暴雨形成。年 

最大洪峰主要出现在4～9 月，锋面雨和台风雨形成的洪水各 占一半。锋面雨洪水多发生在 $4 \sim 7$ 月，洪水特征峰低形胖， 以复峰居多，洪水过程一般历时 2～5 天。台风雨洪水多发生 在 6～9 月，洪水特征暴涨暴落，峰形尖瘦，多为单峰形，洪 水过程一般历时 2～3 天。 

滩坑电站枢纽工程为 I 等工程，主要建筑物拦河坝、溢 洪道、泄洪洞和电站进水口为 1 级建筑物；厂房、引水隧洞、 下游出口消能等均为 2 级建筑物；生态小机永久性水工建筑 物为 3 级建筑物。拦河坝为钢筋混凝土面板堆石坝，相应 1 级建筑物洪水标准为 1000 年一遇设计，可能最大洪水(PMF) 校核；2 级建筑物洪水标准为 100年一遇设计，500 年一遇校 核。坝址区地震基本烈度＜Ⅵ度，考虑到本工程库大坝高， 以及下游有青田县城和温州市，拦河坝设计按Ⅶ度设防。拦 河坝为混凝土面板堆石坝，坝顶高程 171m，最大坝高为 162m， 坝顶长 507m。溢洪道位于左岸垭口，采用开敞式溢洪道，总 长约 1500m，溢流堰堰顶高程 148.00m，设有 6 个闸孔，最大 泄流量 $1 4 3 3 5 ~ \mathrm { ~ m } ^ { 3 } / \mathrm { ~ s ~ }$ 。泄洪洞由施工导流洞改建而成，位于左 岸，洞内设有1 个闸孔，最大泄流量 $1 7 2 9 \mathrm { ~ m } ^ { 3 } / \mathrm { s }$ ，兼作放空水 库之用。引水系统和发电厂房布置于右岸。各枢纽建筑物具 体尺寸和最终设计时的各种特征水位、防洪、兴利效益指标， 见表 5.2-1 工程特性表。

# 1.2.3 气象预报、水雨情监测及洪水预报调度

北海水电公司与浙江专业气象服务有限责任公司签订 《北海水电公司 2024-2025 年滩坑流域专业气象服务合同》， 服务内容主要包括：提供滩坑流域长、中、短期气象预报； 提供重要天气报告；提供气象特别咨询服务；提供汛期降雨 量预报、重要灾害性气候预报等。 

水情自动测报系统控制区间为滩坑水电站坝址以上流域， 中心站、中继站、遥测站按照 1：3：16 的规模建设，中心站 设在浙江浙能北海水力发电有限公司办公楼五楼信息机房， 中继站 3 个分别是仰天湖中继站、北山排中继站、坝上第三 中继，遥测站 16 个包括设遥测雨量站 12 个，设遥测水位雨 

量站 3 个，设遥测水位站 1 个。滩坑水电站水情自动测报系 统采用超短波和卫星通讯混合组网通信方式将其水情信息快 速传输到中心站，各测站点来报正常。站点分布见图5.1-2 滩 坑流域水文遥测站网图。 

滩坑水电站洪水预报方法有两种，分别为人工单位线预 报和新安江模型自动预报，防汛值班人员综合两种预报结果 进行洪水预报。单位线预报由起始 $\mathrm { W } _ { 0 }$ 和时段累积雨量在降水 径流关系图上查得累积径流量，相隔一个时段的两个累积径 流量之差为该时段的径流量；流域稳定下渗率定为 $2 . 0 \mathrm { m m } /$ 时 段，时段径流量减去稳定下渗率则为时段地面径流量；根据 实际的降水情况判别流域降水是否均匀，若不均匀，则根据 降水中心位置来确定采用单位线；采用编制程序进行汇流计 算求出洪峰流量、洪峰出现时间和洪水过程线。新安江模型 自动预报结合滩坑流域特性，根据实时洪水与历史洪水误差 的相似性特征，采用异联想记忆技术、遥测降雨抗差估计技 术与相似性理论三步误差综合修正模式，进行实时洪水预报。 2024 年洪水预报情况见表1.2-3。

# 1.2.4 交通、通讯、电力

滩坑水电站距青田县城西门 $3 2 \mathrm { k m }$ ，距温州市公路里程

# 1.3.4 调度运用情况

1.3.4.1 核定滩坑水电站 2024 年控制运用计划主要内容 

根据浙江省水利厅文件《浙江省水利厅关于滩坑水库 2024 年度控制运用计划的批复》（浙水运管函〔2024〕5 号） 批复，2024 年核定滩坑水电站控制运用计划主要内容为： 

# 一、控制蓄水位

（一）梅汛期、非汛期控制蓄水位 $1 6 0 . 0 0 \mathrm { m }$ （同正常蓄 水位），台汛期控制蓄水位 $1 5 6 . 5 0 \mathrm { m }$ 。除调洪等特殊情况外， 不得超控制蓄水位运行。 

（二）梅台过渡期逐步降低库水位，接省、市气象部门 预报有台风、暴雨影响时，必须及时将库水位降至 $1 5 6 . 5 0 \mathrm { m }$ 。 

# 二、调度原则

# （一）汛期洪水调度

按照《瓯江干流洪水调度方案》（浙防指〔2022〕11号） 有关规定执行，根据预报实施预泄、与瓯江干流洪水错峰调 度。 

1．预报有大暴雨或洪水，洪水预报和会商分析结果研判 库水位将超过控制蓄水位时，通过预泄适当降低库水位。 

2．库水位在控制蓄水位至 $1 6 1 . 5 0 \mathrm { m }$ （20年一遇洪水位） 时，与瓯江干流洪水实行错峰调度，下泄控制鹤城站流量不 超过 $1 4 0 0 0 \mathrm { m } ^ { 3 } / \mathrm { s } \ : ( 2 0$ 年一遇流量），确保青田县城防洪安全； 预报滩坑坝址至鹤城站区间洪水将达到或超过 $1 4 0 0 0 \mathrm { m } ^ { 3 } / \mathrm { s }$ ， 水库关闸，为青田县城减轻防洪压力。

# 2.2 大坝安全评估

# 2.2.1 大坝安全状况

# 2.2.1.1 政府主管部门鉴定检查情况

国家能源局大坝安全监察中心于2017年3月～2018年6 月完成了滩坑水电站大坝安全首次定期检查，专家组评定滩 坑水电站大坝为正常坝，安全等级为 A级。 

根据《关于印发浙江滩坑水电站第三次大坝安全注册检 

查意见的通知》（坝监安监函〔2023〕83号）经国家能源局 大坝安全监察中心现场检查和专家评审，并上报国家能源局 批复，同意滩坑大坝安全注册登记换证，注册等级为甲级， 有效期为 2023 年 4 月 1 日至 2028 年 3 月 31 日。 

根据国家能源局大坝安全监察中心第六轮水电站大坝安 全定期检查规划和 2024 年度计划，滩坑水电站大坝安全第二 次定期检查第一次会议于 2024 年 7 月23 日-25 日现场召开， 并形成会议纪要《滩坑大坝第二次定期检查第一次会议的函》 （坝监安监函〔2024〕283 号）。北海水电公司将根据会议 纪要要求抓紧落实各项工作，确保定期检查工作的顺利进行。 

# 2.2.1.2 大坝安全监测数据情况

截止2024年12月，大坝坝基最大沉降量为 $5 9 4 . 4 4 \mathrm { m m } ( $ （测 点 JV2-2）较去年增加 $0 . 3 7 \mathrm { m m }$ ，坝体内部垂直位移 $9 6 6 . 7 \mathrm { m m }$ （测点 105V1-2），沉降已趋于收敛状态，坝基沉降状态和 性态正常；最大坝体内部水平位移累计水平位移量为 126.4mm（测点 57H2-1），位移值均较小。面板顶部测点最 大水平位移左右岸方向为 $4 2 . 2 \mathrm { m m }$ （测点 LD1-11，向左岸位 移），上下游方向为 $9 2 . 0 \mathrm { m m }$ （测点 LD1-11,向下游位移）， 最大沉降 $2 0 2 . 2 \mathrm { m m }$ （测点 LD1-8）；各位移测点沉降量在纵 向分布上与相对坝高呈正相关，且与相对坝高基本对称。各 周边缝累计最大开合为 $1 4 . 2 2 \mathrm { m m }$ （测点 JR4）、最大剪切为 12.53mm（测点 JR2）、最大沉降为 61.49mm（测点 JR11）； 面板垂直缝变位主要为气温、库水位及时效引起的变化，最 大开合为 24.26mm（测点 J3-1），数据表明面板周边缝及垂 

直缝变位相对稳定，砼面板状况正常。坝后量水堰渗漏量测 值主要随降水量等因素变化而变化，目前大坝渗流量一般在 35.88L/s～61.21L/s 范围内变化，2024 年大坝最大渗流量约 61.21L/s，渗流水水质清澈，未发现有异常情况。其余项目监 测数据也均在合理范围内且满足各项指标要求。 

# 2.2.1.3 库区安全情况

根据相关规程规范及上级部门要求，每年汛前、汛后等 时间段，我公司均组织技术人员进行库区巡查。详查情况如 下：水库干流区域未发现明显的新增库区塌岸、裂缝、滑移， 及不稳定库岸，未发现明显的库水外渗通道，库区无大的水 质污染源；库区交通通畅；未发现库区征地线以下围垦造田 情况、移民线高程以下的非法居住情况。 

滩坑水电站蓄水以来虽有局部滑塌，但库区地质条件总 体较好，近坝库岸稳定。 

# 2.2.2 大坝安全评估结论

结合政府管理部门检查鉴定结论和各项监测数据成果， 表明滩坑水电站大坝等水工建筑物及其边坡运行状况良好， 目前处于稳定安全运行状态。 

# 2.3 汛期划分

我省汛期为每年的 4 月15 日至 10 月15 日，根据水库的 地理位置、暴雨成因及雨型差异，将汛期分为梅汛期和台汛 期，梅汛期一般为 4 月 15 日至 6 月 30 日，台汛期一般为 7 月 16日至 10 月 15日。梅汛期到台汛期的过渡期为 7 月1 日 至7月15日。具体出梅时间以浙江省气象台发布的信息为准。

# 2.5 防洪调度及运用

根据华东勘测设计研究院《滩坑水电站运行设计报告》， 水库正常运行的洪水调度原则如下： 

$\textcircled{1}$ 水库水位台汛期（7～10 月）在汛限水位 $1 5 6 . 5 \mathrm { m }$ 以 上、梅汛期（4～6 月）和枯水期（11～次年3 月）在正常蓄 水位 $1 6 0 . 0 \mathrm { m }$ 以上时开始泄洪。 

$\textcircled{2}$ 台汛期（7～10 月）水库水位在汛限水位 $1 5 6 . 5 \mathrm { m } \sim$ $\mathrm { P = } 5 \%$ 防洪高水位 $1 6 1 . 5 \mathrm { m }$ 之间，即 $1 5 6 . 5 \mathrm { m } < Z { \le } 1 6 1 . 5 \mathrm { m }$ 时，按 补偿凑泄方式调度： 

根据下游青田县城的防洪标准，水库水位在防洪高水位 （ $\mathrm { P = } 5 \%$ ）161.5m 以下时，控制滩坑水库下泄流量，并考虑 下泄流量传播至鹤城站的时间，可使不同洪水组合情况下， 滩坑水库下泄流量与滩坑～鹤城站区间流量之和（包括大溪 干流洪水流量）不超过 $1 4 0 0 0 \mathrm { m } ^ { 3 } / \mathrm { s }$ 。滩坑坝址洪水传播至鹤城 站的时间按 5.0h 考虑。 

即滩坑水库的下泄流量按下式确定： 

$$
Q _ {\text {泄}} = Q _ {\text {安 全}} - D Q _ {1}
$$

Q安全 下游鹤城站断面处的安全泄量 $\left( \mathbf { m } ^ { 3 } / \mathbf { s } \right)$ 。 

DQ1— —考虑洪水传播时间后的滩坑～鹤城站区间流量（包 括大溪干流洪水流量） $( \mathrm { m } ^ { 3 } / \mathrm { s } )$ ）。 

$Q _ { \mathrm { \# \mathbb { L } } }$ — 机组发电流量（ $\mathrm { m } ^ { 3 } / \mathrm { s }$ ）。 

$Q _ { \mathrm { i 1 4 } }$ — 滩坑水库下泄流量（ $\mathbf { m } ^ { 3 } / \mathbf { s }$ ）。 

机组发电流量包含于水库下泄流量之中，调洪计算时机 组发电流量按 $4 0 0 \mathrm { m } ^ { 3 } / \mathrm { s }$ 计，同时滩坑 $\sim$ 鹤城站区间洪水流量 的预报误差考虑 $6 \%$ 左右。 

$\textcircled{3}$ 梅汛期（ $4 \sim 6$ 月）和枯水期（11～次年 3 月）水库水 位在正常蓄水位 $1 6 0 . 0 \mathrm { m } \sim \mathrm { P } { = } 5 \%$ 防洪高水位 $1 6 1 . 5 \mathrm { m }$ 之间，即 $1 6 0 . 0 \mathrm { m } < Z \le 1 6 1 . 5 \mathrm { m }$ 时，按补偿凑泄方式调度。 

$\textcircled{4}$ 水库水位在 $\mathrm { P = } 5 \%$ 防洪高水位 $1 6 1 . 5 \mathrm { m }$ 及以下时，采用 补偿凑泄方式调度，实际调度时的最小下泄流量为机组发电 流量。为了避免瞬间加大溢洪道下泄流量对下游产生过大的 冲击，因此，当水库水位大于防洪高水位 $1 6 1 . 5 \mathrm { m }$ ，且不超过 $1 6 1 . 7 \mathrm { m }$ ，即 $1 6 1 . 5 \mathrm { m } < \mathrm { Z } { \leq } 1 6 1 . 7 \mathrm { m }$ 时，水库按 $6 0 0 0 \mathrm { m } ^ { 3 } / \mathrm { s }$ 控制流 量下泄洪水。 

$\textcircled{5}$ 当水库水位超过 $1 6 1 . 7 \mathrm { m }$ ，即 $Z > 1 6 1 . 7 \mathrm { m }$ 时，则溢洪道 闸门全部打开下泄洪水，电站机组也参与泄洪，即： $Q _ { \mathbb { H } } = Q _ { \mathbb { H } } + Q _ { \mathbb { H } }$ 。 

$\textcircled{6}$ 当水库水位大于 $\mathrm { P { = } } 0 . 2 \%$ 洪水位 $1 6 5 . 2 7 \mathrm { m }$ （即 $Z >$ $Z _ { \mathrm { P } } { = } 0 . 2 \%$ ）时，溢洪道闸门全部打开下泄洪水，并且开启泄洪 洞参与泄洪，发电暂停，即： $Q _ { \mathbb { H } } = Q _ { \mathbb { H } } + Q _ { \mathbb { i } \mathbb { H } }$ 。
